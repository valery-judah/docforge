# Decisions Template

## Purpose
Use this file to capture workstream-local decisions as they happen.

## When To Use
Add an entry when a decision changes the implementation approach, scope, or risk profile.

## Starter Structure
```md
# Decisions

## YYYY-MM-DD - [Decision Title]
- Decision: what was chosen
- Rationale: why this option won
- Impact: what changes because of it
- Follow-ups: remaining actions or checks
- Elevate to ADR: yes | no | maybe
```
