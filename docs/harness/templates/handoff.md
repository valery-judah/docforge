# Handoff Template

## Purpose
Use this file when work pauses, transfers owners, or ends with remaining follow-up.

## When To Use
Update it before stepping away from active work or when handing execution to another person or agent.

## Starter Structure
```md
# Handoff

## Current State
What is done and what is still in motion?

## Remaining Work
- Item 1
- Item 2

## Risks
- Risk 1

## Open Questions
- Question 1

## Next Recommended Actions
1. First action
2. Second action
```
