# Spike Playbook

## When To Use
Use for short investigations, feasibility work, or option analysis.

## Expected Docs To Touch
- Workstream index
- Decisions if the spike leads to a recommended approach
- Evidence with findings, tradeoffs, and references

## Minimum Evidence Expected
- Questions investigated
- Findings and tradeoffs
- Recommendation or next decision point

## Typical Exit Criteria
- Unknowns are reduced enough to unblock the next step
- Recommendation is recorded
- Durable conclusions are promoted to ADRs if needed
