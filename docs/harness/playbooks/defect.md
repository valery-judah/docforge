# Defect Playbook

## When To Use
Use for bug fixes, regressions, and correctness investigations.

## Expected Docs To Touch
- Workstream index
- Evidence with reproduction and fix validation
- Decisions only if the remediation changes approach or scope

## Minimum Evidence Expected
- Reproduction summary
- Fix summary
- Tests or checks that prove the bug is addressed

## Typical Exit Criteria
- Defect is reproduced or bounded
- Fix is validated
- Residual risk is documented if the root cause is only partially addressed
